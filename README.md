# Snake3DGame

3D “Snake / Python racing” game built with React + TypeScript + Vite + Three.js.

## Run

```bash
npm install
npm run dev
```

## Controls

- **Steer**: A/D or ←/→
- **Speed**: W/S or ↑/↓
- **Pause**: Space
- **Camera**: C
- **Restart**: R (when paused or game over)

On touch devices the UI shows an on-screen joystick + buttons.

## Tech

- React + TypeScript + Vite
- Three.js (WebGL)

## Build

```bash
npm run build
npm run preview
```
